# nm-net-strongest

## ARCHIVED

This was an experiment to be honest, to see how comfortable I felt writing C and
GLib code. Turned out, it's not that bad. I still prefer Rust and other
languages, but I enjoyed this experience.

Now, it's archived, I actually wrote this for my college use, so I could get
NetworkManager to scan periodically for strong APs on 5GHz band, as the IT techs
at college had configured the APs funny, where I wasn't getting the strongest
signal. I probably should have gone for an event-based approach rather than
polling.

You live and learn.
